# Easybasket-User

This is a simple User Management REST API built using Spring Boot.
It demonstrates a clean layered architecture with validation, DTO usage, and exception handling.

Features it has:
Create User,
Get All Users,
Get User by ID,
Update User,
Delete User,
Input Validation using Jakarta Validation,
Global Exception Handling,
DTO-based response

Validation Rules applied are :
Name-
  Required,
  Max 20 characters
  
Email-
  Required,
  Must be valid format
  
Password-
  8–20 characters,
  
  Must contain:
  Uppercase letter,
  Lowercase letter,
  Number,
  Special character
